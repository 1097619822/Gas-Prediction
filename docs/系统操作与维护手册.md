# 智慧油气田生产预测与诊断系统 - 操作与维护手册

本手册旨在指导用户如何运行、演示及维护基于 `src_v2` 架构的毕业设计系统。该系统集成了 **PSO-LSTM 产量预测**、**专家规则积液诊断**以及 **Streamlit 可视化大屏**。

---

## 一、 环境准备
在首次运行前，请确保安装了所有核心依赖库：
```bash
pip install pandas numpy torch scikit-learn matplotlib seaborn openpyxl statsmodels streamlit plotly -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

## 二、 系统核心工作流
系统采用模块化设计，建议按照以下顺序执行脚本：

### 1. 产量预测终极实验 (`main_ultra.py`)
*   **功能**：执行数据清洗、特征提取，运行 **PSO 优化的双向 LSTM** 模型，并产出精度指标。
*   **命令**：`python main_ultra.py`
*   **产出**：控制台输出 RMSE/R2 指标，弹出预测对比曲线。

### 2. 积液诊断自动标注与训练 (`main_diagnosis.py`)
*   **功能**：基于物理判据（专家系统）对所有井进行状态标注，并训练积液分类模型。
*   **命令**：`python main_diagnosis.py`
*   **产出**：保存 `final_diagnosis_model.pth` 权重文件。

### 3. 全量数据汇总导出 (`export_summary.py`)
*   **功能**：扫描全部 31 口井，统计每口井的平均产量、积液频率等关键指标。
*   **命令**：`python export_summary.py`
*   **产出**：`processed_results/全井生产汇总分析表.xlsx`（用于论文附录）。

## 三、 Web 演示大屏操作

### 1. 启动 Web 服务
请在项目根目录（`src_v2`）执行以下命令：
```bash
python -m streamlit run visualization/web_app.py
```
*   **访问方式**：启动后，在浏览器访问 `http://localhost:8501`。
*   **演示亮点**：
    *   **交互性**：左侧切换井号，右侧图表实时更新。
    *   **预警逻辑**：当诊断状态为 `Accumulation` 时，仪表盘会自动变红并发出预警。

### 2. 后台启动（Windows）
如果你希望服务在后台静默运行：
```powershell
Start-Process python -ArgumentList "-m streamlit run visualization/web_app.py --server.headless true" -WindowStyle Hidden
```

### 3. 关闭 Web 服务
*   **交互模式下**：在命令行窗口按下 `Ctrl + C`。
*   **强制关闭（包含后台进程）**：
    ```powershell
    taskkill /F /IM python.exe /T
    ```

## 四、 模块解耦与扩展说明
*   **`pre_processing/`**：若以后有新的数据格式，只需修改 `data_loader.py` 中的索引映射。
*   **`models/`**：若想引入 Informer，可在 `web_app.py` 中增加对应的模型加载分支。
*   **`config.py`**：管理所有全局路径，严禁在其他脚本中写死物理路径。

---

## 五、 常见报错解决
1.  **KeyError / TypeError**：通常是 Excel 编码乱码引起，`DataLoader` 已通过索引补位解决。
2.  **Missing Glyph 警告**：这是缺少中文字体，不影响计算，仅影响本地图片弹出时的中文显示（Web 大屏不受影响）。
3.  **ArrowInvalid**：Streamlit 传输数据类型不一致，已在 `web_app.py` 中通过 `.astype(str)` 修复。
